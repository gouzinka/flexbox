CSS _flexbox_

When To Use Flexbox

What is it good for

CSS flexible box layout is a technique for content-driven layout, with an effective distribution of space based on unitless element sizes. This method was designed to manipulate items inside a container on smaller scale - think navigation, forms and so on. It's also your solution for element centering (or general aligning) of any kind.


Didn't understand a word I said? Go ahead and have a look at the specification.

  • Distributing space based on elements content
  • Creating Equal Height Columns
  • Reordering Elements
  • Horizontal And Vertical Centering
  • (Creating Fully Responsive Grids)
  • Easily solve common layout issues
    eg. Sticky footer
What is it not good for

Flexbox layout was designed for manipulating elements on small-scale layouts, so it falls behind when used for larger-scale. But fear not, you can tackle this one with CSS Grid.

Real-world Examples

Properties Used With Flexbox

Defining flex container

We use display properties flex or inline-flex to define a flex container (parent element) which holds its direct children - flex items.

.container {
	display: flex;
	display: inline-flex;
}
Setting items direction

flex-direction is used to control the direction in which are flex-items placed inside their container. We can lay them either horizontally or vertically and even change their order in their respective main-axis.

.container {
	flex-direction: row |
			row-reverse |
			column |
			column-reverse;
}
Wrapping content

Forcing the content of flex container to stay at one line or allowing it to "overflow" onto multiple lines is simply solved with flex-wrap property.

.container {
	flex-wrap: nowrap |
		   wrap |
		   wrap-reverse;
}
Content flow

There is a shorthand property flex-flow used for specifying flexible items direction and whether they should wrap or not.

.container {
	flex-flow: flex-direction flex-wrap;
}
Aligning content (main axis)

The distribution of items along the main axis of the container can be defined by a property justify-content.

.container {
	justify-content: flex-start |
			 flex-end |
			 center |
			 space-between |
			 space-around |
			 space-evenly;
}
Aligning content (cross axis)

In a perpendicular direction to main axis, with align-items property, we can specify space distribution of flex items as well.

.container {
	align-items: flex-start |
		     flex-end |
		     center |
		     baseline |
		     stretch;
}
Aligning lines of content (cross axis)

With align-content property, we define distribution of lines of content (if there is more than one).

.container {
	align-content: flex-start |
		       flex-end |
		       center |
		       space-between |
		       space-around |
		       stretch;
}

Properties for flex-items

Item - order

By setting number to a order property, we can change the position of each one of the flex items.

.item {
	order: <integer>;
}
Item - grow

flex-grow property enables us to change the distribution of items to non-equal proportions.

.item {
	flex-grow: <integer>;
}
Item - shrink

flex-shrink creates the possibility for items to shrink if needed.

.item {
	flex-shrink: <integer>;
}
Item - base size

flex-basis property defines the initial size of an element.

.item {
	flex-basis: <width> |
		    auto;
}
Item - flex

flex property is used for excluding item of being displayed as a flex-item. It is also a shorthand for flex-grow, flex-shrink and flex-basis properties.

.item {
	flex: none |
	      [<flex-grow> <flex-shrink> <flex-basis>];
}
Item - align

align-self overrides alignment value set by the flex container for individual item.

.item {
	align-self: auto |
		    flex-start |
		    flex-end |
		    center |
		    baseline |
		    stretch;
}
